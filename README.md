# Bank Loan Analysis

## Project Description
This project analyzes bank loan data to assess loan performance, customer risk profiles, and approval trends. The analysis supports better credit decision-making and risk management.

## Tools Used
- Excel (Data Cleaning & Validation)
- SQL (MySQL) – Data Analysis
- Power BI – Dashboard & Visualization

## Key KPIs
- Total Loan Amount
- Loan Approval Rate
- Default Rate
- Average Interest Rate
- Loan Status Distribution

## Key Insights
- Identified factors influencing loan approval and rejection
- Analyzed default trends across customer segments
- Highlighted high-risk and low-risk borrower profiles
- Provided insights to improve loan portfolio quality

## Files Included
- Dataset
- SQL Queries
- Power BI Dashboard
- Excel Analysis
